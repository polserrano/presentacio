# 📘 Portfoli de Projectes — CFGM SMX

Benvinguts al meu repositori final del cicle formatiu de grau mitjà de **Sistemes Microinformàtics i Xarxes (SMX)**.

Aquest repositori recopila tots els projectes, pràctiques i activitats més importants que he desenvolupat al llarg de la meva formació acadèmica. L’objectiu principal és mostrar l’evolució dels meus coneixements tècnics, les competències adquirides i la manera de treballar durant aquests dos anys de formació.

Durant el curs he treballat en diferents entorns relacionats amb:

- 💻 Administració de sistemes
- 🌐 Xarxes i configuració de serveis
- 🐧 Sistemes operatius Windows i Linux
- 🔒 Seguretat informàtica
- 🛠️ Muntatge i manteniment d’equips
- ☁️ Virtualització i serveis en xarxa
- 📂 Documentació tècnica i resolució d’incidències

Cada projecte inclou documentació, configuracions, captures, scripts o fitxers relacionats amb les activitats realitzades.
