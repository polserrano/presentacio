# 💡 Projecte 05 — La Incubadora

Aquest projecte consistia en la creació d’una startup tecnològica dins d’una incubadora d’empreses simulada. L’objectiu principal era combinar tots els coneixements tècnics treballats durant el cicle amb conceptes d’emprenedoria, viabilitat econòmica, organització empresarial i desenvolupament professional.

A diferència dels projectes anteriors, aquí no només era important la part tècnica, sinó també la capacitat de detectar problemes reals, plantejar una solució viable i defensar-la davant d’un possible client o inversor. Durant el projecte vam haver de construir una proposta empresarial coherent, definint el model de negoci, els serveis oferts, els costos i la manera de presentar la idea de forma professional.

També es va treballar el desenvolupament del perfil professional personal, reflexionant sobre les habilitats pròpies, els objectius laborals i el rol que cada membre podia assumir dins una empresa tecnològica.

A nivell tècnic, es van reforçar coneixements relacionats amb Git i GitHub Classroom, així com aspectes relacionats amb llicenciament de sistemes operatius i sostenibilitat tecnològica mitjançant una visió Green IT.

El projecte finalitzava amb una presentació tipus “Shark Tank”, on s’havia de defensar la idea empresarial davant d’un tribunal simulant una ronda d’inversió real.
